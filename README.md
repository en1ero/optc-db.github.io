# OPTC Character Database

Character database for One Piece Treasure Cruise.

This is an isolated character database featuring comprehensive character stats, abilities, evolutions, and filtering options for One Piece Treasure Cruise.

This project is written in HTML and JavaScript, using AngularJS, Bootstrap, and jQuery.

# Features

* **Character Table** - Browse all OPTC characters with advanced filtering
* **Search** - Search by name, ID, or aliases
* **Filters** - Filter by type, class, stars, cost, and custom matchers
* **Character Details** - View detailed information including:
  - Captain abilities
  - Sailor abilities
  - Special abilities
  - Evolution chains
  - Drop locations
  - Tandem abilities
  - Pirate Rumble stats

# Running Locally

This is a static site. To run it locally:

**Option 1: Python HTTP Server**
```bash
python -m http.server 8000
```

**Option 2: Node.js http-server**
```bash
npm install -g http-server
http-server -p 8000
```

**Option 3: VS Code Live Server**
- Install the "Live Server" extension
- Right-click `index.html` and select "Open with Live Server"

Then open `http://localhost:8000` in your browser.

**Note:** Character images are loaded from external sources, so an internet connection is required to view them.
